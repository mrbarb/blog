How to install a minecraft (bedrock edition) in Docker

I will make tutorials for just java and java and bedrock in the future (with geyser) but for now, we're going to focus on bedrock.

The first step:

check if git is installed (you should have docker installed, if not follow this tutorial) 

 sudo apt install git

Then we're going to copy an pretty good install script:

 git clone https://github.com/karlrees/docker_bedrockserver

After that we're changing directory in to the cloned github code

cd docker_bedrockserver

and run the script for one server (if you have an error try chmod 755 setup_standalone.sh) 

  sudo ./setup_standalone.sh

Thos Blog just covers the veery basics to get you started, defently check the repo for more detailed info how to configure everything or run more servers at once.
